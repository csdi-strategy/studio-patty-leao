# ✂️ Studio Patty Leão - Plataforma Digital Integrada
> **Desenvolvido por CSDI STRATEGY** | *"O futuro não improvisa."*

## 💻 Sobre o Projeto
Este repositório contém o código-fonte e a documentação técnica da plataforma digital desenvolvida para o **Studio Patty Leão**. 
O objetivo principal é substituir a gestão manual e improvisada por uma solução tecnológica autônoma, focada na automação operacional, registro de clientes, vitrine digital e controle de insumos.

## 🎯 Os 7 Pilares Estratégicos
O desenvolvimento desta solução está estruturado em 7 eixos fundamentais de entrega:
1. **Site Estático:** Interface principal e presença digital da marca.
2. **Banco de Dados:** Modelagem e estruturação para armazenamento seguro (foco em clientes, produtos, estoque e agendamentos).
3. **BI Simples:** Dashboards para inteligência de negócio e tomada de decisão.
4. **Agendamentos:** Módulo de automação de horários e serviços.
5. **Vitrine Digital:** Exposição clara e atrativa do portfólio do salão.
6. **Controle de Estoque:** Gestão automatizada da entrada e saída de insumos.
7. **Usabilidade Mobile:** Interface responsiva, priorizando o acesso via smartphone.

## 🛠️ Tecnologias e Ecossistema
Para garantir a **Estabilidade** e o **Resultado**, o projeto utiliza as seguintes tecnologias e ferramentas:
* **Desenvolvimento Web:** HTML, CSS, JavaScript (e frameworks definidos por Sprint).
* **Banco de Dados:** Modelagem Relacional (DB Designer) / Implementação Não Relacional (NoSQL).
* **Ambiente Local:** Apache / XAMPP.
* **Integrações:** Consumo de APIs (___a definir____).
* **Versionamento:** Git e GitHub.

## ⚙️ Governança e Metodologia Ágil
A gestão do projeto segue o framework **Scrum**, com adaptações para o modelo acadêmico/extensionista:
* **Sprints:** Ciclos semanais (Sprints 1/1) focados em entregas contínuas.
* **Dailies e Alinhamento:** Comunicação ativa via Discord e Microsoft Teams.
* **Gestão de Tarefas:** Backlog e fluxo de operações centralizados no Trello (Quadros de Governança e Operações).
* **Tomada de Decisão:** Maioria simples, com foco no destravamento de impedimentos técnicos.
